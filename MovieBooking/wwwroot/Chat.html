© 2018 - Packt Movie Booking- The code is provided as is without warranties and is for educational and learning purpose only. To keep things simple and comprehensive, focus is on ASP.NET Core and not on design of movie booking. This is over simplification of single screen cinema hall for easy implementation.So, the app assumes that:

  • This is a single screen Cinema hall which shows only one movie at a time.
  • Booking is done for the current day only (Time check is not in place, but can be easily implemented)
  • Seat selection would be done by the backend, and not by the user. Only number of seats would be provided by user.

The purpose of this app is to demonstrate the following:

  1. Getting started with EF Core. How to setup and use EFCore
  2. Introduction to Razor pages, which I feel should enhance the productivity in developing UI.
